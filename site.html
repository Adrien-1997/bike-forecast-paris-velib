FAQ

La démo met quelques secondes à démarrer, normal ?

Oui, c’est le cold start de Cloud Run. Les accès suivants sont instantanés. Vous pouvez augmenter l’instance minimum pour éviter ce délai.

Puis‑je intégrer l’app dans mon site ?

Oui, via un simple <iframe>. La page gère déjà le responsive, le thème et le focus clavier.

Comment sont calculées les prévisions ?

Le modèle apprend les dynamiques horaires par station et incorpore des signaux calendrier/météo. Une baseline de persistance sert de repère pour mesurer l’amélioration.

Intégration Cloud Run

Remplacez l’URL ci‑dessous par l’endpoint public de votre service NiceGUI.

<iframe
  src="https://YOUR-CLOUD-RUN-URL.example/"
  width="100%" height="68vh" style="border:0"
  allow="fullscreen"></iframe>

Vous pouvez aussi placer l’app derrière un sous‑domaine (ex. app.votredomaine.fr) et gérer le SSO côté proxy.
