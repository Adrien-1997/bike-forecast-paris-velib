What you’ll find

  • API FastAPI endpoints for serving & monitoring artifacts
  • UI Next.js frontend (maps + charts)
  • Jobs Pipelines producing JSON artifacts (monitoring / serving)
  • Data Open data + weather, exported and compacted over time

Run locally

See README-local.md in the repository for the full setup. Minimal commands: 

# API (local)
.\.venv\Scripts\python.exe -m uvicorn api.app:app `
  --reload `
  --reload-dir api `
  --port 8081 `
  --env-file api/.env

# UI (local)
cd ui
npm install
npm run build
npm run start
